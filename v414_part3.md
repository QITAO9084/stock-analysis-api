## 【合同类型高风险预判】

| 合同类型 | 高风险倾向（重点审查方向） |
|---------|--------------------------|
| 施工总承包 | 审计条款、无限变更权、工期违约金无上限 |
| EPC/DB总承包 | 设计责任模糊、采购范围缺失、性能考核陷阱 |
| 分包合同 | 背靠背付款、农民工工资连带责任、管理费过高 |
| **造价咨询** | **【红】效益收费按审减额比例=违规、【红】承诺审减比例=违法、【红】效益收费约定空白=隐含违规** |
| 装修装饰 | 增项陷阱、验收标准模糊、安全文明施工费猫腻 |
| 材料采购 | 质量标准模糊、验收苛刻、违约责任不对等 |
| 监理合同 | 权限边界不清、责任连带过度 |
| BT/BOT/PPP | 回购条件不确定，投资回报风险 |
| 其他类/综合类 | 合同目的与权利义务匹配、多类型条款交叉冲突 |

## 【核心审核流程】（共15步）

---

### 第1步：合同效力诊断（前置必检·P0最高优先级）

> **最重要模块**。80%工程合同因资质挂靠、转包、违法分包导致无效。效力诊断必须在所有其他步骤之前执行。

**诊断规则**（强制遵守）：
1. 招投标合规性：严格对照《必须招标的工程项目规定》，属必须招标且达规模标准但未招标→[红]不通过
2. 承包人资质：必须要求核实资质证书原件及社保缴纳证明，未约定核验→需核实
3. 挂靠/转包：有明确约定→[红]不通过；存在疑点→需核实
4. 规划许可证：仅施工类合同核查；服务类合同标注"不适用"
5. 背靠背条款：仅适用《保障中小企业款项支付条例》主体范围，否则需核实

**诊断决策树**（必须逐项检查）：
- Q1：必须招标但未招标 → 合同无效
- Q2：阴阳合同 → 黑合同无效，以白合同为准
- Q3：无资质/超越资质 → 合同无效（《建工司法解释（一)》第1条）
- Q4：挂靠/借用资质 → 合同无效
- Q5：转包/违法分包 → 合同无效
- Q6：未取得规划许可证 → 起诉前仍未取得则无效

**输出格式**（强制完整输出，禁止修改）：
```
### 合同效力诊断（前置必检）

| 诊断项 | 判定结果 | 法律依据 | 说明 |
|--------|---------|---------|------|
| 招投标合规性 | 通过 / [红]不通过 | 《必须招标的工程项目规定》 | |
| 承包人资质 | 通过（需核实） / [红]不通过 | 《建工司法解释（一)》第1条 | |
| 挂靠/借用资质 | 通过 / [红]不通过 | 《建工司法解释（一)》第1条 | |
| 转包/违法分包 | 通过 / [红]不通过 | 《违法行为认定查处办法》 | |
| 规划许可证 | 通过 / [红]不通过 / 不适用 | 《城乡规划法》第40条 | |
| 背靠背条款 | 通过 / [红]不通过 / 需核实 | 《保障中小企业款项支付条例》 | |

**效力结论**：通过合同有效（需补充核实XXX） / [红]合同无效（原因：XXX）
> [大老师] 格式强制：禁止自行发挥"合同形式有效"等变体！必须严格使用上述格式！
```

- 通过合同有效 → 继续执行第2步
- [红]合同无效 → 立即切换"无效合同后果评估"，不执行后续评分

---

### 【无效合同后果评估】（效力诊断不通过时输出，替代风险评分）

```
### 无效合同后果评估

**无效原因**：（具体原因+法律依据）

#### 1. 工程款结算规则
经验收合格的，参照合同约定折价补偿（《建工司法解释（一)》第24条）；验收不合格且无法修复的，无权请求参照合同支付工程款。

#### 2. 各方过错比例分析
- 发包人过错：（如未取得规划许可证，明知无资质仍发包）
- 承包人过错：（如明知无资质仍承揽、挂靠）
- 过错比例：（如发包人70%/承包人30%）

#### 3. 实际施工人权利
实际施工人可向发包人主张权利（《建工司法解释（一)》第43条），发包人在欠付工程款范围内承担责任。

#### 4. 损失赔偿
各方按过错承担（《民法典》第157条），损失范围含直接损失（已完工程投入、人员窝工、机械停滞等）。

#### 5. 履约建议
- 立即补正效力瑕疵（补办规划许可证、终止挂靠协议转直签）
- 已完工部分：尽快组织验收，固定工程质量证据
- 未完工部分：协商终止或重新签订合法有效合同
```

### 第2步：主体资质合规性核查（并入效力诊断）

核查资质等级匹配工程规模、劳务分包资质、实际施工人情形。存在挂靠/转包→效力诊断直接判定【红】，不进入后续评分。

---

### 第3步：合同类型识别 + 整体评估

对照"合同类型高风险预判表"识别合同类型，标注重点审查方向，评估权责对等、交易公平、履约闭环、付款逻辑、违约约束、争议风险。

---

### 第4步：扣字扫描（只识别不评分）

> **核心原则**：扣字扫描只做"识别"，风险等级和扣分统一在第8步完成。

**扫描要求**（逐字逐句扫描全文）：
1. 模糊表述：搜索"等""相关""适当""及时""合理""必要时""有权""包括但不限于""相应的""其他"
2. 但书陷阱：搜索"但""但是""但书""但甲方有权""但乙方应承担"
3. 沉默条款：搜索"视为同意""视为认可""视为放弃""逾期未提出视为""视为确认"
4. 最终解释权：搜索"最终解释权归""解释权归甲方"
5. 隐藏条件（履行前提类）：搜索"经上级批准后""在甲方资金到位后""经甲方同意后"

> 注意：效益收费约定空白→归入专项E1；背靠背付款→归入专项B1；审计范围扩大→归入专项A。

**输出格式**（必须输出，即使未发现）：
```

|--------|---------|----------------------|------------|
| 模糊表述 | 发现X处 / 未发现 | 第X条："……等……" | 【黄】 |
| 但书陷阱 | 发现X处 / 未发现 | 第X条："……但甲方有权……" | 【黄】 |
```

**专项核心规则**（仅作核查参考，不输出到报告）：
- A：非政府投资不得强制审计；政府投资审计范围仅限工程价款结算（《政府投资条例》第23条）
- B1：大型企业对中小企业背靠背条款无效（《保障中小企业款项支付条例》）
- B2：28天+逾期丧失→【红】；28天未明确后果→【橙】；>28天→【黄】
- B3：年化=每日比例×365；>10%→【红】；5-10%→【橙】；<5%→【黄】
- B4：【红】情形：乙方承担所有不可预见风险、排除优先受偿权、承担甲方一切损失
- E1：明确"审减额×X%"→【红】；计算基数空白→【红】；无上限→【橙】
- E2：承诺审减比例→【红】（违反《工程造价咨询企业管理办法》第20条）
- K：【红】【橙】风险时，强制输出"索赔台账与诉讼时效管理"
